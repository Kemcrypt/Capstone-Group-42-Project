# Capstone-Group-42-Project
📌 Project Title
Digital Safety Toolkit for Schools: Enhancing Cyber-Hygiene in Education

📌 Project Overview
This project focuses on improving cybersecurity awareness and practices in schools by providing a Digital Safety Toolkit. The toolkit is designed to help students, teachers, and school administrators develop safe digital habits through simple, practical, and affordable tools.

It promotes a secure digital learning environment as education increasingly relies on technology.

📌 Problem Statement
There is a significant cybersecurity awareness gap in schools:

Students and teachers lack basic cybersecurity knowledge

Common risks include:

Phishing attacks

Weak passwords

Unsafe browsing

Data breaches

👉 These risks negatively impact both learning and safety in digital environments.

📌 Solution
The project proposes a Cyber-Hygiene Toolkit, which is:

Simple

Affordable

Scalable

🔧 Toolkit Components:
Password Managers (e.g., Bitwarden)

Browser Extensions (e.g., HTTPS Everywhere, uBlock Origin)

Privacy Checklists for safe online behavior

Training Guides for schools

📌 SDG Alignment
Aligned with the United Nations Sustainable Development Goals:

SDG 4 – Promotes safe and inclusive digital learning

SDG 9 – Encourages digital infrastructure awareness

SDG 16 – Supports safe and secure environments

📌 Technologies Used
Cybersecurity tools (password managers, browser extensions)
